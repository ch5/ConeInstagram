Clone Simple apps instagram
===========================
This is simple example integrating rails 5 and Bootstrap 
CRUD implementation with photos.

### Features gem ###
1. masonry-rails
2. haml
3. bootsrap3
4. devise
5. simple-form
6. paperclip
7. acts-as-votable
